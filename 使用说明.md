# 网站使用说明

## 🚀 快速启动

由于浏览器安全限制（CORS），网站需要通过本地服务器访问才能正常加载账号数据。

### 方法一：使用启动脚本（推荐）

1. **双击** `start-server.sh` 文件
   - 或者在终端中运行：`./start-server.sh`

2. 浏览器会自动打开，或手动访问：**http://localhost:8000**

3. 停止服务器：按 `Ctrl + C`

### 方法二：手动启动 Node.js 服务器

```bash
# 1. 打开终端（Terminal）

# 2. 进入网站目录
cd ~/Desktop/ai-website/shadowrocket.best

# 3. 启动 Node.js 服务器（推荐）
node server.js

# 4. 在浏览器访问
# http://localhost:8000
```

### 方法三：Python 服务器（备选，功能受限）

```bash
cd ~/Desktop/ai-website/shadowrocket.best
python3 -m http.server 8000
```

**注意**：Python 服务器无法解决 CORS 问题，账号数据可能无法加载。建议使用 Node.js 服务器。

## ❓ 常见问题

### Q: 为什么不能直接双击 index.html 打开？
A: 浏览器的安全策略阻止 file:// 协议访问外部API，必须通过 http:// 协议访问。

### Q: 为什么推荐使用 Node.js 服务器？
A: Node.js 服务器内置了 API 代理功能，可以绕过 CORS 限制，确保账号数据正常加载。

### Q: 端口 8000 被占用怎么办？
A: 编辑 `server.js` 文件，修改第 5 行的 `PORT = 8000` 为其他数字（如 8080、3000 等）

### Q: 服务器启动后怎么停止？
A: 在终端窗口按 `Ctrl + C`

### Q: 账号数据多久更新一次？
A: 页面每 5 分钟自动刷新一次数据，也可以手动刷新浏览器页面。

### Q: 显示"无法连接到服务器"怎么办？
A:
1. 确保使用 `./start-server.sh` 或 `node server.js` 启动服务器
2. 检查终端是否有错误信息
3. 确认访问的是 `http://localhost:8000` 而不是直接打开文件
